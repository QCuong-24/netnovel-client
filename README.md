# NetNovel Client

React + TypeScript client for reading, uploading, and managing novels or mangas.

## Foundation

- Vite + React + TypeScript
- Tailwind CSS + shadcn-style UI primitives
- React Router layouts
- TanStack Query + Axios API client
- English/Vietnamese i18n
- Light/dark theme support
- Reader settings foundation for font, text size, width, and reading background
- Ad slot foundation for future ad providers

## Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` type-checks and builds the production bundle.
- `npm run lint` runs ESLint.
- `npm run preview` serves the production build locally.

## Environment

Copy `.env.example` when you need local overrides.

```env
VITE_APP_NAME=NetNovel
VITE_API_BASE_URL=http://localhost:8080
```
